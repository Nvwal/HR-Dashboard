# HR Analytics Dashboard

An interactive Power BI dashboard designed to monitor and analyze key insights from human resources data. This dashboard focuses on workforce demographics, compensation overview, and turnover trends to support data-driven HR decisions.

✨ Key Features:
1-Dynamic KPIs: Displays real-time critical metrics such as Total Employees, Average Salary, and Attrition Rate paired with trend sparklines.

2-Demographic Breakdown: Visualizes employee distribution by Sex, Marital Status, Citizenship, Race, and Age Groups using clean, modern donut and bar charts.

3-Advanced Navigation & UX: Implements a sleek custom sidebar for seamless page switching (Overview, Attrition, Employees Details) and utilizes Modern Visual Tooltips with integrated Drill-Through capabilities for granular data exploration.

4-Professional UI/UX: Styled with a polished dark/light contrast theme to maximize scannability and visual hierarchy.
